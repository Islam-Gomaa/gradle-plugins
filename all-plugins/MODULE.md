# Module accessors

Set of all `javiersc` plugins

### Download from MavenCentral

```kotlin
// buildSrc/build.gradle.kts

dependencies { 
    implementation("com.javiersc.gradle-plugins:all-plugins:$version") 
}
```

### Usage

It contains all `javiersc` Gradle plugins.

### Samples

```kotlin
plugins {
    `javiersc-docs`
}
```
